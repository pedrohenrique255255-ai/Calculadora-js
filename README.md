# Calculadora Simples

Este projeto é uma calculadora simples desenvolvida utilizando **HTML, CSS e JavaScript**.  
Ela permite realizar operações matemáticas básicas de forma rápida e direta no navegador.

## Funcionalidades

A calculadora realiza as seguintes operações:

- Soma
- Subtração
- Multiplicação
- Divisão

As operações são executadas utilizando **funções em JavaScript**, enquanto o **HTML** estrutura a interface e o **CSS** é responsável pela estilização.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do Projeto

```
calculadora/
│
├── index.html
├── style.css
└── script.js
```

- **index.html** → estrutura da calculadora  
- **style.css** → estilização da interface  
- **script.js** → lógica e funções da calculadora
